### 接触的 linux 命令

| 命令                                       | 解释                               |
| ------------------------------------------ | ---------------------------------- |
| lsof -i:\<port\>                           | 检查端口占用.                      |
| kill \<pid\>                               | 根据 pid 杀死进程，有选项          |
| unzip \<file\> -d \<dst\>                  | 解压 zip 文件到相应位置            |
| ssh <username>@<IP address or domain name> | ssh 连接                           |
| logout                                     | ssh 断开                           |
| cp/mv \<file\> \<dst\>                     | 复制、移动(重命名) file -> dst     |
| vmstat 1 5                                 | 每秒采集一次 cpu 使用率，采集 5 次 |
